# priya-birthday

Folder structure:
- `index.html` = intro page
- `surprise.html` = main content
- `css/intro.css`
- `css/surprise.css`
- `js/intro.js`
- `js/surprise.js`
- `assets/images/`
- `assets/audio/`

Replace the empty placeholder files inside `assets/images/` and `assets/audio/` with your real files:
- photo1.webp
- photo2.webp
- photo3.webp
- letter.webp
- music.mp3
